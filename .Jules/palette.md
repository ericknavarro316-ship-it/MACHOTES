## 2025-02-14 - Interactive Treeview Keyboard Accessibility and Visual Affordance
**Learning:** By default, Tkinter's `Treeview` (often used inside CustomTkinter for tabular data) relies primarily on mouse interactions (like `<Double-1>`). Text-based "checkboxes" like `[X]` are common but lack clear visual affordance. Adding spacebar bindings (`<space>`) to toggle state and upgrading brackets to universal unicode checkboxes (`☑` and `☐`) significantly improves keyboard accessibility and visual clarity without requiring complex custom widget implementations.
**Action:** When working with tabular data that acts as a selection list, proactively replace ASCII text brackets with unicode checkboxes and bind the `<space>` key to the toggle action using `tree.selection()`. Ensure getter methods check for the new unicode values while safely supporting legacy data if needed.
